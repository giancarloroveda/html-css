Tags em HTML5

    
Em HTML5 podemos adicionar conteúdo em uma página
por meio de tags que indiquem qual é o sentido
daquele conteúdo dentro do site. Por isso, o HTML5
é chamado, também, de HTML semântico.



Passe o mouse por cima dos itens abaixo para descobrir como realizar tais funções!
  • Criar parágrafos
  • Criar títulos
  • Criar imagens
  • Criar abreviações
  • Criar códigos-fonte
  • Criar listas numeradas
  • Criar listas com demarcadores
